# HeroSpin

## Clone the repo
git clone git@github.com:SamiAljord/HeroSpin.git

## Set up
Run the following commands:
```shell
$ yarn install
$ yarn ios-gems
$ yarn pods
```

## Running the app on iOS:

```shell
$ yarn ios
```
